# 高尔夫网站
## 开发环境
webStorm
## 页面要求：响应式布局

导航栏统一（除了场馆方入口）：Logo+如下<br>
* 登录/注册  
* 首页		
* 高尔夫产品		
* 论坛	
* 场馆方入口	
* 个人空间（登陆后）	
* 我的资料
我的订单
消息99+
群消息99+
VIP服务 
## 主要技术点
* 响应式布局（CSS媒体查询（Media Query），bootstrap框架）
* 首页滚动特效（监听滚动条事件）
* 论坛富文本编辑
* 三级联动
* 地址动态选择
* 产品查看的放大镜效果（仿淘宝）
* ajax实现用户登录注册重置
